# Companion Orb — Web Preview

Open `preview/companion-orb/index.html` in your browser to see an interactive preview of the orb UI.

Recommended (serve locally):

```bash
# from repository root
python3 -m http.server 8123 --directory preview/companion-orb
# then open http://localhost:8123
```

Controls are provided for drive mode, mood, RPM, sleeping state and simple reactions.
